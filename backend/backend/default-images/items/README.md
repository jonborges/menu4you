# Imagens de Itens (Categorias)

Coloque aqui as imagens das categorias de produtos:

- `acai.jpg` - Categoria Açaí
- `bebidas.jpg` - Categoria Bebidas
- `brasileira.jpg` - Categoria Brasileira
- `doces.jpg` - Categoria Doces
- `fitness.jpg` - Categoria Fitness
- `japonesa.jpg` - Categoria Japonesa
- `lanche.jpg` - Categoria Lanche
- `pizza.jpg` - Categoria Pizza

Essas imagens serão automaticamente atribuídas aos itens baseado na categoria selecionada.

**URL de acesso:** `http://localhost:8080/default-images/items/[nome-arquivo].jpg`

**Exemplo:** `http://localhost:8080/default-images/items/pizza.jpg`
